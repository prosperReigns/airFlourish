5️⃣ Transport
Admin creates transport
curl -X POST http://127.0.0.1:8000/transports/admin/ \
-H "Authorization: Token <ADMIN_TOKEN>" \
-H "Content-Type: application/json" \
-d '{
  "transport_name": "Airflourish Van",
  "vehicle_type": "van",
  "pickup_location": "Lagos Airport",
  "dropoff_location": "Victoria Island",
  "price_per_passenger": 2000.00,
  "currency": "NGN"
}'
User books transport
curl -X POST http://127.0.0.1:8000/transports/booking/ \
-H "Authorization: Token <USER_TOKEN>" \
-H "Content-Type: application/json" \
-d '{
  "transport_id": 1,
  "passengers": 3,
  "special_requests": "Need child seat"
}'
Transport search
curl -X GET "http://127.0.0.1:8000/transports/search/?pickup=Lagos&dropoff=Victoria&date=2026-03-15" \
-H "Authorization: Token <USER_TOKEN>"
6️⃣ Visa
Apply for visa
curl -X POST http://127.0.0.1:8000/visas/ \
-H "Authorization: Token <USER_TOKEN>" \
-H "Content-Type: application/json" \
-d '{
  "flight": 1,
  "destination_country": "United Kingdom",
  "visa_type": "Tourist",
  "appointment_date": "2026-03-10"
}'
Approve visa (Admin)
curl -X POST http://127.0.0.1:8000/visas/approve/1/ \
-H "Authorization: Token <ADMIN_TOKEN>" \
-H "Content-Type: application/json" \
-d '{"action": "approve"}'
Reject visa (Admin)
curl -X POST http://127.0.0.1:8000/visas/approve/2/ \
-H "Authorization: Token <ADMIN_TOKEN>" \
-H "Content-Type: application/json" \
-d '{"action": "reject"}'

/register/	POST	❌	None	Register new user
/profile/	GET	✅	IsAuthenticated	Get user profile
/logout/	POST	✅	IsAuthenticated	Logout user

/bookings/	GET	✅	Authenticated	List bookings (admin: all, user: own)
/bookings/	POST	✅	Authenticated	Create new booking
/bookings/{id}/	PUT/PATCH	✅	Admin only	Update booking status
/bookings/cancel/{id}/	POST	✅	Authenticated	Cancel booking
/bookings/create/	POST	✅	Authenticated (10/min throttle)	Generic booking creation
/bookings/flights/search/	GET	✅	Public/Auth	Search flights (Amadeus integration)

/flights/flights/	GET	✅	Authenticated	List flight bookings
/flights/flights/	POST	✅	Authenticated	Create flight booking
/flights/secure-book/	POST	✅	Authenticated	Initiate flight payment
/flights/verify-payment/	POST	✅	Authenticated	Verify & confirm flight booking

/hotels/	GET	✅	Authenticated	List hotels
/hotels/	POST	✅	Admin Only	Create hotel
/hotels/{id}/	PUT/PATCH	✅	Admin Only	Update hotel
/hotels/{id}/	DELETE	✅	Admin Only	Delete hotel
/hotel-reservations/	POST	✅	Authenticated	User books hotel
/hotel-reservations/	GET	✅	Authenticated	List reservations

/admin-transports/	GET/POST	✅	Admin Only	Admin manage transport services
/transport-options/	GET	✅	Authenticated	List available transport
/transports/	POST	✅	Authenticated	User books transport
/transport-reservations/	GET/POST	✅	Authenticated	Manage reservations
/admin-transport-reservations/	GET	✅	Admin Only	Admin view all reservations
/search/	GET	✅	Authenticated	Search by pickup/dropoff/date

/visas/	GET	✅	Authenticated	List user's visa applications
